# uragaki-ui

PoC開発専用のペーパープロトタイピングライクUI

shadcn/ui と rough.js をベースにした手書き風UIコンポーネントライブラリです。
プロトタイピング、ワイヤーフレーム作成、遊び心のあるアプリケーションに最適なスケッチスタイルのコンポーネントを提供します。

---

## 事前準備

```bash
pnpm install
```

---

## 使用方法

```bash
# 開発サーバー起動（ブラウザで確認しながらコンポーネント編集）
pnpm dev

# Storybook 起動（コンポーネントカタログを確認）
pnpm storybook

# ライブラリビルド（npm公開用）
pnpm build:lib

# ドキュメントサイト + Storybook をビルドして GitHub Pages にデプロイ
pnpm deploy:pages

# npm にパッケージを公開
pnpm publish:npm <version>
#   version: patch | minor | major | x.y.z
```

---

## 特徴

- **手書き風ボーダー** — rough.js の SVG レンダリングによるスケッチスタイル
- **手書き風フォント** — Architects Daughter、Patrick Hand、Caveat
- **紙テクスチャ背景** — グリッドパターン付き
- **アクセシビリティ** — shadcn/ui（Radix UI）ベースの完全対応
- **Tailwind CSS v4** によるスタイリング
- **ダークモード対応**
- **安定したシード値レンダリング** — 描画のちらつきを防止

---

## インストール（利用者向け）

---

## アーキテクチャ

各 Uragaki コンポーネントは、対応する shadcn/ui コンポーネントを `UragakiBorder` オーバーレイでラップしています。
UragakiBorder は rough.js を使って手書き風の SVG シェイプを `pointer-events: none` の絶対配置オーバーレイとして描画し、
元のコンポーネントの操作性を完全に維持します。
